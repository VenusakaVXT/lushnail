COURSE CURRICULUM

DETAILED COURSE CONTENT

6 Sessions (4 weeks - 38 hours)

Introduction & Basic Knowledge

  • Overview of nail profession, working process in salon

  • Hygiene and safety when doing nails

  • Understanding nail structure, common nail conditions

  • Proper technique for holding clippers, files, and tools

  • Dry practice to familiarize with techniques on practice hand

Foundation Polish Techniques

  • Review basic knowledge and familiarize with common base polish types.

  • Practice even color application, handling polish streaks and common errors on real nails.

  • Techniques for light and dark base polish, simple ombre and harmonious color matching.

  • Complete practice of base polish set on real model.

Creating Shapes & Basic Art

  • Guide to creating standard nail shapes: square, round, oval, almond.

  • Basic gel/powder application techniques to increase durability and correct nail imperfections.

  • Practice simple designs: lines, dots, basic flowers and leaves.

  • Apply shapes and designs to complete nail set for clients.

Practical Process & Skill Completion

  • Practice complete client service process from greeting to consultation.

  • Practice a complete nail set according to simulated client requirements under instructor guidance.

  • Share experience on opening a salon, managing clients and building personal brand.

  • Course summary, Q&A session and career development roadmap guidance.
